## 📌 Atividade 4 – Nome completo

**O que faz:**  
Garante que o usuário digite nome e sobrenome corretamente.

**Como funciona:**  
- O programa pede o nome e o sobrenome  
- Verifica se algum dos dois está vazio  
- Se estiver vazio, pede novamente  
- Repete até que ambos sejam preenchidos  
- Depois, exibe o nome completo  

**Ideia do código:**
```javascript
while (nome == "" || sobrenome == "")

Importante:
O uso do || (OU) faz o programa repetir se qualquer um dos campos estiver vazio.
