# 🔁 Atividades com `while`

## 📌 Atividade 1 – Número positivo
**O que faz:**  
Pede um número até o usuário digitar um valor válido (positivo).

**Como funciona:**  
- O programa pede um número  
- Verifica se ele é negativo  
- Se for, pede novamente  
- Repete isso até o número ser positivo  

**Ideia do código:**
```javascript
while (numero < 0)
